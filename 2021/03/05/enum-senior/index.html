java 枚举类高级使用

前言

枚举是一种规范它规范了参数的形式，这样就可以不用考虑类型的不匹配并且显式的替代了int型参数可能带来的模糊概念 枚举像一个类，又像一个数组。Enum作为Sun全新引进的一个关键字，看起来很象是特殊的class, 它也可以有自己的变量，可以定义自己的方法，可以实现一个或者多个接口。 在涉及到使用大量需要标记的内容的时候,我们一般选择使用枚举类型来快速定义内容,但是不仅仅是因为快速,同时使用枚举类型更加的安全。下面先看两个典型的枚举类的例子：
public enum Direction{EAST,SOUTH,WEST,NORTH};
public enum Position{LEFT,RIGHT,UP,DOWN};
实际上这就是声明就是一个类，它们刚好有四个实例。（注意在比较枚举类型的值时。永远不要调用equals。直接用==即可,我们有时需要将两个枚举类进行关联改怎么去使用呢，比如一年有四个季度，每季度对应不同的月份，这时候我们可以创建两个枚举类，一个是季度的枚举类，一个是月份的枚举类
