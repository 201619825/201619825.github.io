使用hexo+github+live2d快速搭建博客

准备工作

  • 安装Node.js：http://nodejs.cn/download/
  • 安装Git：http://git-scm.com/downloads
  • 准备github账号：https://github.com/

github邮箱需校验、node，git安装目录不要包含中文

github操作步骤

  1. 点击右上角头像，选择new repository
  2. repository name以[你的用户名.github.io]命名
  3. 创建一个index.html，随便输入，在浏览器端打开[你的用户名.github.io]，成功显示

github ssh配置

  1. 全部配置用户名和邮箱

  2. 点击右键，选择git bash here，输入以下命令生成密钥

  3. 将生成的公钥添加至github

    • 生成的key一般保存在你的家目录下的.ssh里[C:\Users\你的电脑设备名.ssh]
    • 打开id_rsa.pub–全选复制
    • 打开github–点击右上角–setting–SSH and GPG keys–New SSH key–粘贴进去即可

安装hexo [hexo中文文档]https://hexo.io/zh-cn/

  1. 在D盘(磁盘随意选)新建一个blog(文件名随意)目录，进入该目录，右键–git bash here

  2. hexo初始化(warn无视，error重视)

hexo发布至github(属性值的key的冒号后一定要间隔一个空格)

hexo主题使用

  • 打开hexo官方主题模板https://hexo.io/themes/
  • 看上哪个主题直接点进去跟着教程走即可
  • 下载后的主题放置本地博客目录的themes文件夹下即可
  • 应用主题样式，修改博客根目录下的_config.yml的theme属性
