解决@Value不能给静态变量赋值

前言. @Value注解使用

1、@Value注解必须要在spring的bean中才能使用，不能自己new一个对象调用

2、@Value只能给普通变量赋值，不能给静态变量赋值

3、@Value是在AbstractAutowireCapableBeanFactory类的doCreateBean方法中进行初始化，即交由Spring容器进行处理，如果没有@Component及其衍生注解注释Spring无法识别，导致无法获取到配置值。

4、把@Value(value=”${properties.alias}”)放到静态变量的set方法上面即可，需要注意的是set方法要去掉static，还有就是当前类要交给spring来管理
